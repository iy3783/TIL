2. 클라우드 보안 위협
기술적인 측면의 위협:
-가상화로 인한 보안 문제: 가상화로 인해 기존 보안 문제 방어 힘들어지고 
파급 효과 커짐

-하이퍼바이저 감염 위험
-가상머신 공격 경로:가상 네트워크를 통한 패킷 스니핑
-공격자의 익명성 : 공격자 누군지 파악 힘듬: 하이퍼 바이저 안의 가상화 스위치를 통해
외부에서의 패킷을 가상 os 들로 보낸다. => 하이퍼바이저와 가상화 스위치는 블라인드존이다.
블라인드존(security blind zone)은 볼 수가 없다.
-가상머신의 이동성으로 인한 문제
:가상화 환경에서는 물리적 플랫폼간에 VM이동이 쉽다. =>감염 확산 문제 발생

+관리 측면의 문제 + 내부자 문제+해커들 타겟 + 피해규모의 확산 + 법/제도적인 문제

=> lastpass : saas 비밀관리하는 제품 

-접근 및 인증 : FIDM (Federated identity management): 사용자 ID를 제 3의 인증기관
에서 관리=> 휴대폰 인증번호
